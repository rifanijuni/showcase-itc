# ITC SHOWCASE ✨

PROJECT digunakan untuk
memenuhi seleksi THE HUNTER Web Development ITC UPNYK

Tema : “Power Full Front-End Website For ITC”

ITC Showcase adalah front-end website yang dirancang untuk mendokumentasikan dan menampilkan berbagai proyek dari seluruh divisi di Information Technology Club (ITC) UPN Veteran Yogyakarta. Website ini memungkinkan pengguna untuk melihat, menambah, mengedit, dan menghapus data.

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- Javascript
- Node.js & NPM


## Run Locally

Clone project

```bash
  git clone https://github.com/rifanijuni/showcase-itc.git
```

Masuk ke direktori project

```bash
  cd showcase-itc
```

Pastikan Node.js dan NPM sudah terinstal di komputer. Kemudian, install dependencies dengan perintah:

```bash
  npm install
```

Jalankan server lokal dengan perintah:

```bash
  npm start
```
Proyek akan otomatis terbuka di browser bawaan

```bash
  http://127.0.0.1:8080/
```
